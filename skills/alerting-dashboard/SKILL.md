# Alerting Dashboard Skill

Use this skill when configuring or reviewing monitoring dashboards and alert policy.

## Workflow

1. Identify what should be monitored and why.
2. Register targets only after confirming names, transport, URLs, or pipeline metadata.
3. Configure alert thresholds only after explicit approval.
4. Review dashboard and report output.
5. Document alert policy, owners, thresholds, and known gaps.
